# nodeRISE
========

Create a nodeJS server has become simple!


## Getting started with NodeRISE
  
  $ `cd ~/`
  
  $ `git clone git://github.com/johnrlive/noderise.git`
  
  $ `cd noderise`
  
  $ `sh noderise-setup.sh`
